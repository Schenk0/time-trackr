# Time Trackr

A minimal day-level time tracker. Log what you spend each time slot on, view daily stats, and set recurring schedules.

## Features

- **Timeline view** — visualize your day split into 15 or 30 minute slots
- **Tags** — create color-coded tags for activities
- **Schedules** — define recurring time blocks per weekday
- **Stats** — see a daily breakdown of logged time by tag
- **Notifications** — get reminded when a slot goes unlogged
- **Offline** — all data stored in localStorage, no backend needed

## Tech Stack

Next.js, React, TypeScript, Tailwind CSS, shadcn/ui

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
